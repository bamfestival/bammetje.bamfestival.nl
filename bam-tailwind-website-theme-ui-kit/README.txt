BAM! Festival Tailwind Website Theme + UI Kit

Files
- index.html           Full festival landing page demo
- components.html      Isolated component examples
- theme.js             Tailwind CDN theme config for local demos
- tailwind-theme.js    Theme tokens for a real Tailwind project
- README.txt           This file

What is included
- Navbar
- Hero
- Line-up grid
- Timetable
- Buttons, badges, cards
- Info / sponsor CTA blocks
- Footer

Usage
1. Open index.html in a browser for a quick preview.
2. For production Tailwind projects, copy the values from tailwind-theme.js into your own tailwind.config.js.
3. Replace demo copy and acts with BAM! content.

Style direction
- Inspired by modern festival websites
- Warm, high-energy gradients
- Big typography
- Premium dark backgrounds
- Strong CTAs and cards
